# Arena Mod — 竞技场模组

> Minecraft Fabric 1.20.1 | 作者：明天会更好

---

## 简介

Arena Mod 是一个竞技场玩法模组。它创建了一个独立的竞技场维度，玩家通过放置**竞技核心**方块进入模式选择界面，选择**单人塔防**或**多人PVP**两种模式之一开始游戏。

模组包含完整的波次战斗、商人交易、经验经济、奖杯成就和音乐播放系统。

## 单人模式

### 战斗流程

玩家被传送到 80×80 的封闭竞技场，需要抵御一波又一波的怪物攻击。每波包含多个批次，清完一批刷下一批，清完一波进入下一波。通关全部波次后进入下一轮回，怪物强度和数量递增。

### 波次详情

- **第1波** — 5批怪物。僵尸、骷髅、蜘蛛、苦力怕、史莱姆等基础怪物，让玩家熟悉战斗节奏。
- **第2波** — 2批怪物。怪物组合强化，加入女巫等麻烦角色，考验玩家的装备和走位。
- **第3波** — 1批怪物。精英怪物为主，加入烈焰人，需要足够的火力和生存能力。
- **第4波** — 1批怪物。Boss级波次，包含远古守卫者、凋零骷髅等强力敌人，是每轮回的最大挑战。

### 生命与死亡

- 玩家拥有 **30条命**
- 每次死亡消耗1条命，同时失去身上所有物品
- 死亡后会在竞技场中心复活，但装备已丢失
- 命数归零则游戏结束
- 可以用 `/arena life` 指令消耗80经验点购买1条命

### 商人系统

商人通过指令 `/arena individual 114514` 在玩家当前位置召唤。共有三种商人，每种最高3级，升级后解锁更多商品。

**物品商人** — 出售武器、防具、工具、弓箭等装备
- 1级：基础装备
- 2级：进阶装备（升级费50经验）
- 3级：顶级装备（升级费100经验）

**方块商人** — 出售各类建筑方块
- 1级：基础建材
- 2级：中级建材（升级费40经验）
- 3级：高级建材（升级费80经验）

**药水商人** — 出售各类效果药水
- 1级：基础药水
- 2级：进阶药水（升级费60经验）
- 3级：强力药水（升级费120经验）

### 轮回与奖杯

通关全部4波后视为完成一轮回，获得 **1个奖杯 + 200经验点**，然后进入下一轮回继续挑战。奖杯可以通过指令兑换稀有物品。

## PVP模式

### 匹配规则

玩家选择PVP模式后进入匹配队列。**人数没有上限**，2人以上即可开赛，人数越多战况越混乱。匹配成功后所有玩家被传送到PVP竞技场。

### 竞技场环境

PVP竞技场是 **30×30** 的虚空场地，地面是双层基岩（y=0 和 y=1），四周无围墙，上方无天花板。全亮度、永远白天、没有天气。整个场地就是一个悬浮在虚空中的方形平台。

### 战斗规则

- **时间限制**：20分钟倒计时，时间到了按存活情况结算
- **死亡掉落**：被击杀后身上的物品全部掉落在死亡位置，不保留
- **方块限制**：每人最多放置80个方块，超过无法继续放置
- **方块清理**：所有玩家放置的方块每5分钟自动清除一次
- **边界检测**：逃出竞技场范围视为淘汰，物品原地掉落
- **地板机制**：竞技场地板会周期性消失再恢复，掉下虚空直接淘汰

### 胜负结算

- **最后一人生存**：该玩家获胜，获得 **1个奖杯 + 300经验点**，比赛结束
- **时间到多人存活**：平局，所有存活玩家各获得 **50经验点**（不获得奖杯）
- **时间到无人存活**：全员淘汰，无人获得奖励

### 观战模式

被淘汰的玩家自动切换为旁观者模式，可以自由视角观看剩余比赛。竞技场南侧（边界外3格）建有阶梯式观战台，5级石英台阶铺红毯，观众可以在此观战。比赛结束后所有参与者（包括观众）自动归还入场前保存的物品并传送回主世界。

## 命令列表

| 指令 | 作用 | 适用 |
|------|------|------|
| `/arena resume` | 恢复上次保存的游戏进度 | 单人 |
| `/arena new` | 删除旧存档，重新开始 | 单人 |
| `/arena individual 114514` | 在当前位置召唤三名商人 | 单人 |
| `/arena life` | 消耗80经验点购买1条命 | 单人 |
| `/arena trophy exchange <物品> <数量>` | 消耗1个奖杯兑换物品 | 通用 |
| `/arena clearblocks` | 清除所有已放置的方块 | PVP |

## 音乐系统

模组支持在游戏目录下的 `.minecraft/arenamod/music/` 文件夹中放入音频文件，进入竞技场时自动播放，离开自动停止。

支持格式：wav、ogg、mp3、mp4、flac、aac、m4a、wma

播放控制指令：

| 指令 | 作用 |
|------|------|
| `/arena music next` | 下一首 |
| `/arena music previous` | 上一首 |
| `/arena music loop` | 循环当前曲目 |
| `/arena music order` | 顺序播放 |
| `/arena music random` | 随机播放 |
